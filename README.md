### 操作說明
```
ENTER : 開始遊戲
上下左右鍵 : 1p移動飛機加速度方向
WASD鍵 : 2p移動飛機加速度方向
P : 跳出遊戲

黑洞: 黑洞周圍有加速度，會把飛機拉入黑洞。
子彈: 自動沿著飛機頭方向連發。
Time: 倒數3分鐘結束遊戲。
```
